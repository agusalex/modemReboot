# grive2Docke

 - Reboot Fibertel Technicolor modem
# Environmental Variables
| Variable |Description  |
|--|--|
|**URL**  | Modem URL |
|**USER**  | Modem USER |
|**PASS**  | Modem Password |

# Running
- Run this container  with these **ENV** options : 
```docker run -it -e URL=URL -e USER=USER PASS=PASS agusalex/modemReboot ```
